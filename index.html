Cargar grupos

Carga un CSV por grupo. Columnas esperadas: Nombre, Apellido(s), Nombre de usuario, Número de ID, Dirección de correo. Las cabeceras se detectan de forma flexible.

Los datos quedan guardados en este dispositivo. Si recargas la app, no hace falta volver a cargar los CSV.

⚠️ Aviso RGPD / LOPDGDD

El tratamiento de imágenes de documentos de identidad es un dato personal sensible. Lee las advertencias completas en el README antes de usar esta app con alumnos reales. Esta herramienta no almacena la imagen del documento y procesa todo localmente.
